# Αιτιατική (Accusative Case) of Greek Names

The **Αιτιατική** (Aitiatiki) is the accusative case in Greek grammar, used to indicate the direct object of a verb. This document provides comprehensive rules and examples for forming the accusative case of Greek names.

## Overview

The accusative case is one of the five cases in Modern Greek (along with nominative, genitive, vocative, and dative). It is used when a name functions as the direct object of a verb, answering the question "whom?" or "what?". In English, this is similar to the object form: "I see **George**" or "I call **Maria**".

The accusative case is typically used with the articles **τον** (masculine), **την** (feminine), and **το** (neuter).

## General Rules

### Male Names

#### Names ending in **-ος** (os)
- Remove **-ος** and add **-ο**
- **Γιώργος** → **τον Γιώργο**
- **Πέτρος** → **τον Πέτρο**
- **Στέφανος** → **τον Στέφανο**
- **Αλέξανδρος** → **τον Αλέξανδρο**

**Note:** Unlike the vocative case, the accusative formation for -ος names does not depend on stress position. All -ος names follow the same pattern: **-ος → -ο**.

#### Names ending in **-ης** (is)
- Remove **-ης** and add **-η**
- **Γιάννης** → **τον Γιάννη**
- **Δημήτρης** → **τον Δημήτρη**
- **Αντώνης** → **τον Αντώνη**
- **Βασίλης** → **τον Βασίλη**

#### Names ending in **-ας** (as)
- Remove **-ας** and add **-α**
- **Κώστας** → **τον Κώστα**
- **Θανάσης** → **τον Θανάση** (Note: -ης becomes -η)
- **Ανδρέας** → **τον Ανδρέα**

**Note:** Some names ending in **-ας** are actually **-ης** names, so they follow the **-ης → -η** pattern:
- **Θανάσης** → **τον Θανάση** (not τον Θανάσα)

#### Names ending in **-ούς** (ous)
- Remove **-ούς** and add **-ού**
- **Μανώλης** → **τον Μανώλη** (Note: -ης becomes -η)

### Female Names

#### Names ending in **-α** (a)
- Usually remain the same
- **Μαρία** → **την Μαρία**
- **Άννα** → **την Άννα**
- **Σοφία** → **την Σοφία**
- **Αικατερίνη** → **την Αικατερίνη** (Note: -η ending)

#### Names ending in **-η** (i)
- Usually remain the same
- **Ελένη** → **την Ελένη**
- **Αικατερίνη** → **την Αικατερίνη**
- **Κατερίνα** → **την Κατερίνα** (Note: -α ending)

#### Names ending in **-ου** (ou)
- Usually remain the same (these are often genitive forms used as surnames)
- **Παπαδοπούλου** → **την Παπαδοπούλου**

## Detailed Patterns

### Pattern 1: Masculine -ος → -ο
```
Nominative:  Γιώργος, Πέτρος, Στέφανος, Αλέξανδρος
Accusative:  τον Γιώργο, τον Πέτρο, τον Στέφανο, τον Αλέξανδρο
```

**Important:** Unlike the vocative case, the accusative formation for -ος names is consistent regardless of stress position. All -ος names follow the **-ος → -ο** pattern.

### Pattern 2: Masculine -ης → -η
```
Nominative:  Γιάννης, Δημήτρης, Αντώνης, Βασίλης
Accusative:  τον Γιάννη, τον Δημήτρη, τον Αντώνη, τον Βασίλη
```

### Pattern 3: Masculine -ας → -α
```
Nominative:  Κώστας, Ανδρέας
Accusative:  τον Κώστα, τον Ανδρέα
```

**Note:** Some names ending in **-ας** are actually **-ης** names, so they follow the **-ης → -η** pattern:
- **Θανάσης** → **τον Θανάση** (not τον Θανάσα)

### Pattern 4: Feminine Names
Most feminine names remain unchanged in the accusative case:

```
Nominative:  Μαρία, Ελένη, Άννα, Σοφία
Accusative:  την Μαρία, την Ελένη, την Άννα, την Σοφία
```

## Special Cases and Exceptions

### Diminutive Forms
Diminutive forms follow the same patterns as their base forms:

- **Γιωργάκης** → **τον Γιωργάκη**
- **Νικολάκης** → **τον Νικολάκη**
- **Μαρία** (diminutive: **Μαριώ**) → **την Μαριώ**

### Compound Names
For compound names, typically only the first name changes:

- **Γιώργος Παπαδόπουλος** → **τον Γιώργο Παπαδόπουλος**
- **Μαρία Κωνσταντίνου** → **την Μαρία Κωνσταντίνου**

### Names with Particles
When using names with particles (του/της), the particle usually remains:

- **Γιώργος του Παπά** → **τον Γιώργο του Παπά**
- **Μαρία της Ελένης** → **την Μαρία της Ελένης**

### Definite Article Agreement
The accusative case requires the appropriate definite article:
- **Masculine:** **τον** (singular), **τους** (plural)
- **Feminine:** **την** (singular), **τις** (plural)
- **Neuter:** **το** (singular), **τα** (plural)

## Usage in Context

### Examples in Sentences

1. **"Βλέπω τον Γιώργο."** (I see George.)
2. **"Φωνάζω την Μαρία."** (I call Maria.)
3. **"Συναντώ τον Νίκο."** (I meet Nikos.)
4. **"Καλώ την Ελένη."** (I call Eleni.)
5. **"Αγαπώ τον Δημήτρη."** (I love Dimitris.)
6. **"Επισκέπτομαι την Σοφία."** (I visit Sophia.)
7. **"Θυμάμαι τον Κώστα."** (I remember Kostas.)
8. **"Γνωρίζω την Άννα."** (I know Anna.)

### With Prepositions

The accusative case is also used with certain prepositions:

- **"Για τον Γιώργο"** (For George)
- **"Με την Μαρία"** (With Maria)
- **"Χωρίς τον Νίκο"** (Without Nikos)
- **"Προς τον Δημήτρη"** (Toward Dimitris)

### Formal vs. Informal

In formal contexts, especially with titles, the accusative may be used with the title:

- **"Βλέπω τον Κύριο Γιώργο"** (I see Mr. George) - formal
- **"Συναντώ την Δρ. Μαρία"** (I meet Dr. Maria) - formal with title
- **"Βλέπω τον Γιώργο"** - informal

## Rules Summary

### Quick Reference

**Masculine Names:**
- **-ος** → **-ο** (Γιώργος → τον Γιώργο)
- **-ης** → **-η** (Γιάννης → τον Γιάννη)
- **-ας** → **-α** (Κώστας → τον Κώστα)

**Feminine Names:**
- Usually remain unchanged (Μαρία → την Μαρία)

**Important Notes:**
- The accusative formation for -ος names is **consistent** (unlike vocative) - always **-ος → -ο**
- The accent (τόνος) position may change in the accusative
- Always use the appropriate article: **τον** (masculine), **την** (feminine)
- Some names have irregular accusative forms
- Diminutive forms follow their base name patterns

## Comparison with Other Cases

### Accusative vs. Genitive vs. Vocative

| Case | Masculine -ος | Feminine -α | Usage |
|------|---------------|-------------|-------|
| **Nominative** | Γιώργος | Μαρία | Subject |
| **Genitive** | του Γιώργου | της Μαρίας | Possession |
| **Accusative** | τον Γιώργο | την Μαρία | Direct object |
| **Vocative** | Γιώργε | Μαρία | Direct address |

## Implementation Notes

When implementing accusative case conversion in code:

1. **Check the ending** of the name first
2. **For names ending in -ος:** Always replace -ος with -ο (no stress dependency)
3. **For names ending in -ης:** Replace -ης with -η
4. **For names ending in -ας:** Replace -ας with -α (but check if it's actually -ης)
5. **For feminine names:** Usually no change needed
6. **Add the appropriate article:** τον (masculine), την (feminine)
7. **Preserve the accent** position correctly
8. **Test with common names** to ensure accuracy

### Example Algorithm

```
IF name ends in "-ος":
    accusative = name[:-2] + "ο"
    article = "τον"
ELSE IF name ends in "-ης":
    accusative = name[:-2] + "η"
    article = "τον"
ELSE IF name ends in "-ας":
    IF name is actually "-ης" pattern (e.g., Θανάσης):
        accusative = name[:-2] + "η"
    ELSE:
        accusative = name[:-2] + "α"
    article = "τον"
ELSE IF name is feminine:
    accusative = name (usually unchanged)
    article = "την"
```

## Related Cases

For reference, here are the other Greek cases:

- **Ονομαστική** (Nominative): The subject form - "Γιώργος"
- **Γενική** (Genitive): The possessive form - "του Γιώργου"
- **Αιτιατική** (Accusative): The object form - "τον Γιώργο" ← **This document**
- **Κλητική** (Vocative): The addressing form - "Γιώργε"
- **Δοτική** (Dative): Rarely used in modern Greek
