# Κλητική (Vocative Case) of Greek Names

The **Κλητική** (Klitiki) is the vocative case in Greek grammar, used when directly addressing or calling someone by name. This document provides comprehensive rules and examples for forming the vocative case of Greek names.

## Overview

The vocative case is one of the five cases in Modern Greek (along with nominative, genitive, accusative, and dative). It is used when you are speaking directly to someone, similar to how we use names in English when calling someone: "Hey, George!" or "Maria, come here!"

## General Rules

### Male Names

#### Names ending in **-ος** (os)
The vocative formation for names ending in **-ος** depends on the **stress (τόνος)** position:

**Παροξύτονα** (Paroxytone - stressed on the penultimate syllable):
- Remove **-ος** and add **-ε**
- **Γιώργος** → **Γιώργε** (Γιώρ-γος, stressed on "γιώρ")
- **Πέτρος** → **Πέτρε** (Πέ-τρος, stressed on "πέ")
- **Στέφανος** → **Στέφανε** (Στέ-φα-νος, stressed on "στέ")

**Οξύτονα** (Oxytone - stressed on the last syllable):
- The vocative **remains the same** as the nominative (no change)
- **Νικολός** → **Νικολός** (Νι-κο-λός, stressed on "λός")
- **Ανδρός** → **Ανδρός** (Αν-δρός, stressed on "δρός")

**Note:** Most common Greek names ending in -ος are paroxytone, so they follow the **-ος → -ε** pattern.

#### Names ending in **-ης** (is)
- Remove **-ης** and add **-η**
- **Γιάννης** → **Γιάννη**
- **Δημήτρης** → **Δημήτρη**
- **Κωνσταντίνος** → **Κωνσταντίνε** (Note: -ος ending)

#### Names ending in **-ας** (as)
- Remove **-ας** and add **-α**
- **Κώστας** → **Κώστα**
- **Θανάσης** → **Θανάση** (Note: -ης becomes -η)
- **Αντώνης** → **Αντώνη**

#### Names ending in **-ούς** (ous)
- Remove **-ούς** and add **-ού**
- **Μανώλης** → **Μανώλη** (Note: -ης becomes -η)

### Female Names

#### Names ending in **-α** (a)
- Usually remain the same, but sometimes add **-α** (already ends in -α)
- **Μαρία** → **Μαρία**
- **Άννα** → **Άννα**
- **Ελένη** → **Ελένη** (Note: -η ending)

#### Names ending in **-η** (i)
- Usually remain the same
- **Ελένη** → **Ελένη**
- **Σοφία** → **Σοφία** (Note: -α ending)
- **Αικατερίνη** → **Αικατερίνη**

#### Names ending in **-ου** (ou)
- Usually remain the same (these are often genitive forms used as surnames)
- **Παπαδοπούλου** → **Παπαδοπούλου**

## Stress-Based Rules for -ος Names

The most important rule for names ending in **-ος** is that the vocative formation depends on **where the stress (τόνος) falls**:

| Stress Type | Greek Term | Stress Position | Vocative Formation | Examples |
|------------|------------|----------------|-------------------|----------|
| **Παροξύτονα** | Paroxytone | Penultimate syllable | **-ος → -ε** | Γιώργος → Γιώργε<br>Πέτρος → Πέτρε<br>Στέφανος → Στέφανε |
| **Οξύτονα** | Oxytone | Last syllable | **-ος → -ος** (no change) | Νικολός → Νικολός<br>Ανδρός → Ανδρός |

**Key Point:** Most common Greek names ending in -ος are **paroxytone** (stressed on the penultimate syllable), so they follow the **-ος → -ε** pattern. However, oxytone names (stressed on the last syllable) remain unchanged in the vocative.

## Detailed Patterns

### Pattern 1: Masculine -ος → -ε (Παροξύτονα)
For **paroxytone** names (stressed on the penultimate syllable):
```
Nominative:  Γιώργος, Πέτρος, Στέφανος, Αλέξανδρος
Vocative:    Γιώργε, Πέτρε, Στέφανε, Αλέξανδρε
```

### Pattern 1b: Masculine -ος → -ος (Οξύτονα)
For **oxytone** names (stressed on the last syllable):
```
Nominative:  Νικολός, Ανδρός
Vocative:    Νικολός, Ανδρός (no change)
```

**Important:** The stress position determines the vocative form. Most common Greek names ending in -ος are paroxytone.

### Pattern 2: Masculine -ης → -η
```
Nominative:  Γιάννης, Δημήτρης, Αντώνης
Vocative:    Γιάννη, Δημήτρη, Αντώνη
```

### Pattern 3: Masculine -ας → -α
```
Nominative:  Κώστας, Θανάσης
Vocative:    Κώστα, Θανάση
```

**Note:** Some names ending in **-ας** are actually **-ης** names, so they follow the **-ης → -η** pattern:
- **Θανάσης** → **Θανάση** (not Θανάσα)

### Pattern 4: Feminine Names
Most feminine names remain unchanged in the vocative case:

```
Nominative:  Μαρία, Ελένη, Άννα, Σοφία
Vocative:    Μαρία, Ελένη, Άννα, Σοφία
```

## Special Cases and Exceptions

### Diminutive Forms
Diminutive forms often have their own vocative patterns:

- **Γιωργάκης** → **Γιωργάκη**
- **Νικολάκης** → **Νικολάκη**
- **Μαρία** (diminutive: **Μαριώ**) → **Μαριώ**

### Compound Names
For compound names, typically only the first name changes:

- **Γιώργος Παπαδόπουλος** → **Γιώργε Παπαδόπουλος**
- **Μαρία Κωνσταντίνου** → **Μαρία Κωνσταντίνου**

### Names with Particles
When addressing someone with a particle (του/της), the particle usually remains:

- **Γιώργος του Παπά** → **Γιώργε του Παπά**

## Usage in Context

### Examples in Sentences

1. **"Γιώργε, έλα εδώ!"** (George, come here!)
2. **"Μαρία, τι κάνεις;"** (Maria, how are you?)
3. **"Νίκο, περιμένε!"** (Nikos, wait!)
4. **"Ελένη, είσαι έτοιμη;"** (Eleni, are you ready?)
5. **"Δημήτρη, πού πας;"** (Dimitris, where are you going?)

### Formal vs. Informal

In formal contexts, especially with titles, the vocative may be used with the title:

- **"Κύριε Γιώργε"** (Mr. George) - formal
- **"Δρ. Μαρία"** (Dr. Maria) - formal with title
- **"Γιώργε"** - informal

## Rules Summary

### Quick Reference

**Masculine Names:**
- **-ος (παροξύτονα)** → **-ε** (Γιώργος → Γιώργε, Πέτρος → Πέτρε)
- **-ος (οξύτονα)** → **-ος** (Νικολός → Νικολός, no change)
- **-ης** → **-η** (Γιάννης → Γιάννη)
- **-ας** → **-α** (Κώστας → Κώστα)

**Feminine Names:**
- Usually remain unchanged (Μαρία → Μαρία)

**Important Notes:**
- **For names ending in -ος, the stress position is critical:**
  - **Παροξύτονα** (stressed on penultimate): -ος → -ε
  - **Οξύτονα** (stressed on last syllable): -ος → -ος (no change)
- The accent (τόνος) position may change in the vocative
- Some names have irregular vocative forms
- Diminutive forms follow their own patterns
- Always consider the stress pattern of the name

## Implementation Notes

When implementing vocative case conversion in code:

1. **Check the ending** of the name first
2. **For names ending in -ος, check the stress position:**
   - If **oxytone** (stressed on last syllable): vocative = nominative (no change)
   - If **paroxytone** (stressed on penultimate): vocative = replace -ος with -ε
3. **Identify the gender** (if possible) to apply correct rules
4. **Handle special cases** and exceptions
5. **Preserve the accent** position correctly
6. **Test with common names** to ensure accuracy

### Example Algorithm

```
IF name ends in "-ος":
    IF name is oxytone (stressed on last syllable):
        vocative = name (no change)
    ELSE (paroxytone - stressed on penultimate):
        vocative = name[:-2] + "ε"
ELSE IF name ends in "-ης":
    vocative = name[:-2] + "η"
ELSE IF name ends in "-ας":
    vocative = name[:-2] + "α"
ELSE IF name is feminine:
    vocative = name (usually unchanged)
```

**Note:** Determining if a name is oxytone or paroxytone requires checking the stress position. Most common Greek names ending in -ος are paroxytone.

## Related Cases

For reference, here are the other Greek cases:

- **Ονομαστική** (Nominative): The subject form - "Γιώργος"
- **Γενική** (Genitive): The possessive form - "του Γιώργου"
- **Αιτιατική** (Accusative): The object form - "τον Γιώργο"
- **Κλητική** (Vocative): The addressing form - "Γιώργε" ← **This document**
- **Δοτική** (Dative): Rarely used in modern Greek
